# Competitive evaluation contract

A competitive capsule reproduces a scientific mechanism and then exposes a bounded optimization
problem. Passing the reproduction gate is necessary but does not determine the winner.

Each capsule manifest declares:

- a competition mode: independent score or head-to-head;
- a primary metric and its direction;
- qualification checks that must be satisfied before a run can win;
- numerical tie tolerance and ordered tiebreakers;
- a plain-language winner rule.

The private verifier converts raw measurements into a normalized objective score in [0, 1] for
cross-run reporting, but must preserve all raw metrics in check measurements. The suite report ranks
the latest verified attempt for every agent within each capsule. Earlier attempts remain in the
audit table but cannot silently replace the latest attempt.

## Winner norms

A winner must reproduce successfully, satisfy every qualification check, and beat other qualifying
runs on the declared primary metric. Scores within the declared tolerance are tied until an explicit
tiebreaker separates them. If the tiebreakers remain tied, report a tie.

Head-to-head arenas require both fixed reference opponents and a cross-submission tournament. The
fixed field makes scores meaningful before all candidates arrive; the tournament provides the direct
model-versus-model result. Run both sides, multiple deterministic maps or seeds, and enough matches
to report uncertainty.

## Visual evidence

Candidate-selected demonstrations are never the sole evidence. The evaluator renders canonical
visible and hidden episodes from the sealed submission. Every competitive capsule should produce:

- raw score and component metrics;
- deterministic replay or scene state sufficient to rerender;
- a canonical showcase generated by the evaluator;
- median and failure cases, not only the best case;
- numerical overlays that connect the visual result to the score.

A beautiful but invalid artifact cannot qualify. A numerically strong artifact that is visually
misleading should be flagged during human review without inventing a hidden model-judge score.

## Calibration gate

Competitive capsules stay draft until they have a passing reference, repeatability measurements,
at least two meaningful failing mutants, hidden-case coverage, and a reviewed comparison packet.
Difficulty labels are hypotheses until pilot runs locate the empirical capability frontier.

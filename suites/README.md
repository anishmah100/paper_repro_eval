# Suites

A suite is a version-pinned list of reproduction capsules. See
[`docs/suite-format.md`](../docs/suite-format.md).


`visual-research-arcade-v0.yaml` is the draft ten-arena competitive suite. It is version-pinned and
fully materializable so common copying and isolation can be tested before private verifiers are
calibrated. See the [visual arena index](../docs/VISUAL_ARENAS.md) and
[competition contract](../docs/COMPETITIONS.md).
